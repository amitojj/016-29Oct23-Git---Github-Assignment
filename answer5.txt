here are the steps to contibute to any open source project.

1.Fork: Fork the open-source project on GitHub.

2.Clone: Clone your forked repository to your local machine.

3.Create Branch: Create a new branch for your changes.

4.Make Changes: Make and commit your code changes.

5.Push and Pull Request:Push changes to your fork and create a pull request.

6.Discussion: Engage in any feedback or discussion with project maintainers.

7.Merge or Iterate: If approved, changes get merged; if not, iterate.

8.Sync: Keep your fork updated with the original repository.
